FRAMEVECTORNORMS - Norm of frame vectors

Usage

vnorms = framevectornorms(F,L)
vnorms = framevectornorms(F,L,idx)

Input parameters

F Frame definition
L System length
idx Index (or range) of vector(s)

Output parameters

vnorms Vector norms

Description

framevectornorms(F,L) returns 2-norms of vectors of frame F for system length L. The number of vectors in a frame (and the length of the output vector) can be obtained as frameclength(F,L).

framevectornorms(F,L,idx) returns 2-norms of vectors with indices idx. Elements in idx must be in range 1:frameclength(F,L).

Real-valued-input frames

By default, the function returns the norm of vectors used for synthesis. Frames like 'dgtreal', 'filterbankreal' do not contain the ''redundant'' conjugate-symmetric vectors and their synthesis operator is not linear. Therefore the vectors used in synthesis do not have an explicit form and their norm is unspecified. The vectors used for analysis are well defined and can be obtained by passing additional flag 'ana'.
